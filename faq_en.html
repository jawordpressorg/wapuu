How To Add To Wapuu's Archives By Pull Request

  1. Fork this repository. If you have already forked and cloned this repository, go to [3].
    • https://github.com/jawordpressorg/wapuu
  2. Clone YOUR OWN REPOSITORY to your local machine.
    • ex: $ git clone git@github.com:tekapo/wapuu.git
  3. If you have already forked and cloned it, you can use that one.
    • http://qiita.com/xtetsuji/items/555a1ef19ed21ee42873
  4. Checkout to "gh-pages" branch.
    • $ git checkout gh-pages
  5. Modify your Wapuu's image format as below and put it under the "wapuu-archive" directory.
    • Data format: PNG
    • Size: 300x330px
    • Background Transparency: On
    • File Name: customname-wapuu.png (ex: tonkotsu-wapuu.png)
  6. Open index.html and add a new section after the last <!--Each image end-->. 
    <!--Each image start-->
<div class="wapuu-each">
<a href="http://example.com">
<img src="wapuu-archive/XXXXXXX-wapuu.png" alt="">
</a>
<div class="wapuu-name">Name: YOUR_WAPUU_NAME</div>
<div class="wapuu-author">Author: AUTHOR_NAME</div>
</div>
<!--Each image end-->
  7. Commit and Push.
  8. Send a pull request from your repository.
